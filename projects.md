## 1. Movie Recommendation System
- **Members:** Shanu Sahu, Aayush Shrivastava, Abhishek Verma
- **Description:** Building a system to suggest movies to users based on their preferences, involving data collection, preprocessing, model building, and a user interface.

## 2. Smart IIoT Predictive Maintenance System
- **Members:** Prakash Dwivedi, Shubhanshi Tiwari, Aryan Singh
- **Description:** An AI-powered solution to predict industrial machine failures by analyzing real-time sensor data (like temperature and torque) using models like Random Forest and XGBoost, with a FastAPI backend and Streamlit dashboard.

## 3. Car Price Prediction
- **Members:** Anurag Raghuwanshi, Athrav Soni, Abhinav Jain
- **Description:** A machine learning web app to estimate used car prices based on features like year, mileage, and fuel type. It's built with Python libraries and deployed using FastAPI and Streamlit.

## 4. AI Chat Bot
- **Members:** Ritika Upadhyay, Vishakha Sahu, Meenakshi Arya
- **Description:** An interactive, rule-based chatbot developed in Python that mimics human conversation. It identifies keywords to answer questions on basic subjects like Python, AI, and machine learning.

## 5. Asynchronous Distributed Secure Transport (ADST) Protocol
- **Members:** Aryan Pathak, Aditi Sharma, Anshika Sonwane
- **Description:** A high-performance protocol for secure, distributed AI. It authenticates workers, encrypts data using AES-256-GCM, and supports both TCP and UDP transport modes.

## 6. Meal Planner
- **Members:** Ishi Sahu, Tanushri Soni, Muskan Ahirwar
- **Description:** An application using an AI-powered chatbot to give meal suggestions and dietary advice. It includes a "Weekly Analyzer" to track eating habits, with data stored in MongoDB.

## 7. PortGenie - Web3 Portfolio Builder
- **Members:** Janvi Akare, Sambhav Kuril, Ujjawal Satankar
- **Description:** An AI-powered tool to create professional portfolios. It uses decentralized storage (IPFS) for resumes and projects and blockchain (NFTs or Soulbound Tokens) for credential verification.

## 8. Air Quality Index (AQI) Prediction System for Madhya Pradesh Cities Using Machine Learning
- **Members:** Jeet Hemnani, Rohan Kambale, Ansh Shinag
- **Description:** A machine learning model that uses historical AQI data from cities in Madhya Pradesh to predict and classify future AQI values (e.g., Good, Moderate, Poor).

## 9. Gesture-based Presentation Controller
- **Members:** Shivam Singh, Ankur Ghosh, Tarun Patel
- **Description:** A computer vision project to control PowerPoint or PDF slides using hand gestures. It uses Mediapipe for hand detection, OpenCV for camera input, and PyAutoGUI to navigate slides.

## 10. FiMyra
- **Members:** Sudhanshu Tiwari, Vedika Karmarkar, Manan Awadhiya
- **Description:** A health app that uses AI to provide a single score showing how mood, diet, and habits affect well-being, offering personal tips like a personal coach.

## 11. Lost & Found Portal
- **Members:** Riya Choubitkar, Mokshi Devi Rajak, Swati Patel
- **Description:** A web platform to help students and staff locate lost items or report found belongings on campus, aiming to bridge the gap between finders and seekers.

## 12. College Enquiry Chatbot
- **Members:** Chinmay Dubey, Varun Ahirwar, Dhruv Sahu
- **Description:** An intelligent chatbot using natural language processing (NLP) to automatically answer student queries about the college (e.g., courses, fees, timings, faculty).

## 13. Project Management Application
- **Members:** Aanchal Acharya, Pallavi Thakur, Yashika Kaithwas
- **Description:** A full-stack (PostgreSQL, Express, React, Node.js) platform to manage projects within an organization, assign tasks to team members, and send email notifications.

## 14. Retired Mentor (Web Application)
- **Members:** Ankit Nishad, Astha Thakur, Atharv Tamrakar
- **Description:** A web platform designed to connect retired professionals with students or learners who are seeking mentorship and guidance in various fields.

## 15. ML Based Project
- **Members:** Harsh Dhare, Amol Shrivastava, Aditya Yadav
- **Description:** "ML based application"

## 16. Synthexia - A Decentralized Synthetic Data Generator and Market
- **Members:** Dhruv Tiwari, Adarsh Sharma, Aditya Malviya
- **Description:** A decentralized, trustless platform for generating, verifying, and trading synthetic data using GAN/VAE models. It stores data on IPFS and uses a Web3 marketplace.

## 17. AI Powered Credit Risk Assessment
- **Members:** Naman Jaiswal, Ayush Wankhade, Pratima Malviya
- **Description:** A machine learning model that acts as a bank's loan officer, using a real-world dataset to predict if a new applicant will repay their loan (GOOD) or default (BAD).

## 18. Clarity
- **Members:** Nipun Lakhera, Sahil Raikwar, Mo Zaid Sheikh, Shivansh Nigam
- **Description:** An AI-powered code refiner. A developer can paste buggy or inefficient Python code, and the AI model will analyze it and provide a corrected, improved version.

## 19. Email/SMS SPAM Classifier
- **Members:** Subhanker Kumar, Jayprakash Patel, Manish Aangan Patel, Satyanarayan Dev Pandey
- **Description:** An email SPAM classifier that works using the Naive Bayes classification algorithm.

## 20. Automatic Number Plate Recognition (ANPR) System
- **Members:** Yash Sharma, Devendra Singh Bais, Abhishek Kumar, Vikas Kharte
- **Description:** A system using image processing and deep learning to capture vehicle images, detect the number plate, and extract the characters using Optical Character Recognition (OCR).
